# happy-hour-speis-inveiders
Progetto Happy Hour 2025/26 classe 3CI della scuola G. Marconi Pontedera (PI), una riproduzione del classico videogioco Space Invaders
questo progetto formativo mira ad accrescere le nostre competenze non solo informatiche, ma anche di lavorare in una squadra. Questo video gioco verrà presentato ai futuri (rispetto ad ora) giorni dell'Happy Hour 25/26.
Come istallare il gioco:
Per scaricare il gioco bisogna avere clion e source git installati (qualunque tipo di git funziona, source git è consigliato)
dopo di chè si va su <code> nella schermata di github, si copia il link, si apre source git e si preme in alto a sinistra, si incolla il link copiato e si specifica la cartella di download e nome.
quando si è fatto tutto questo si apre clion e si clicka sulla voce "open project", per poi specificare il percorso della cartella scaricata tramite source git.
si apre il progetto in clion e si apre la cartella "libs" -> "lib-bin" e copiare i file .dll, per poi copiarli nella cartella cmake-build-debug creata da clion automaticamente.
in seguito sarà possibile premere il tasto in alto a destra verde vuoto, a forma di triangolo, simile ad un tasto play pause per far partire il gioco.
